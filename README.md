# Landing Page Project – Login & Signup

## Description
This project is a modern and responsive landing page designed with HTML and CSS. It features elegant **login** and **signup** forms with a professional look and feel. The design emphasizes simplicity, responsiveness, and user-friendly interaction, making it suitable for showcasing login/signup interfaces.

**Key Features:**
- **Modern Design & Dark Theme:** Full-screen background image with a clean dark mode layout.
- **Glassmorphism Effect:** Blurred backgrounds for navbar and login/signup boxes.
- **Responsive Layout:** Built with Flexbox, adapting smoothly to different screen sizes.
- **Custom Fonts:** Integrated RobotoMono and OpenSans for modern and readable typography.
- **Interactive Elements:** Hover effects for navigation links and buttons.
- **Styled Forms:** Transparent input backgrounds, white borders, smooth rounded edges, and styled placeholder text.
- **Gradient Buttons:** Submit buttons with gradient backgrounds and hover effects.
- **Footer Design:** Transparent footer with centered text.
- **Clean Code Structure:** Organized HTML and CSS for easy maintenance and customization.

## Pages

### 1. Login Page (`index.html`)
- Input fields: Email, Password
- Submit button: "Continue"
- Link for "Forget Password"
- Responsive design adapting to mobile, tablet, and desktop
- Glassmorphism effect for the login form
- Gradient button with hover effect

### 2. Signup Page (`sigin_up.html`)
- Input fields: First Name, Last Name, Email, Password
- Submit button: "Continue"
- Link for "Forget Password"
- Responsive and visually consistent with the login page

### 3. About Page (`about.html`)
- Description of the project
- Key features and highlights
- Centered headings and paragraph styling
- Glassmorphism effect applied for text container
